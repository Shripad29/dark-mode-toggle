# 🌙 Dark Mode Toggle

A simple and modern **Dark Mode Toggle** built using **HTML, CSS, and JavaScript**.

This project allows users to switch between light and dark themes with a smooth animation and saves the preference using localStorage.

---

## 🚀 Features

- Toggle between Light Mode and Dark Mode
- Smooth animation effect
- Saves user preference using localStorage
- Clean and minimal UI
- Beginner-friendly project

---

## 🛠️ Technologies Used

- HTML
- CSS
- JavaScript

---

## ⚙️ How It Works

1. Click on the toggle switch.
2. The theme changes between light and dark mode.
3. The selected mode is saved in localStorage.
4. On page reload, the saved mode is automatically applied.

---

## 💡 Key Concepts

- JavaScript DOM manipulation
- Event handling
- CSS animations (@keyframes)
- localStorage for saving user preferences
